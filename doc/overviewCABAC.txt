CABAC - an entropy coding method
Abbreviation: Context-Based Adaptive Binary Arithmetic Coding
Originallty, It was designed for H264/AVC. And it also a
successor of H265/HEVC.
It contains 3 main phases:
1. Binarization
2. Context modelling
3. Binary Arithmetic Coding
- Regular phase
- Bypass phase

Input : Syntax Element (SE)
Output: Bitstream
